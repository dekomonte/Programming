# Comandos Linux

```pwd``` - Informa o caminho do diretório atual<br>
```sudo``` - Fazer as coisas com permissão de usuário admin <br>
```sudo adduser``` - Adiciona um usuário <br>
```cp``` - Copia um arquivo <br>
```mv``` - Move um arquivo/Renomear um arquivo <br>
```mkdir``` - Cria uma pasta <br>
```rm``` - Remove um arquivo <br>
```rm -r``` -  <br>
```cd``` - Navegação entre diretórios <br>
```cd /``` - Diretorio raiz<br>
```cd -``` - Diretorio que estava antes<br>
```cd ~``` - Diretorio do usuário<br>
```ls``` - Lista arquivos e pastas do diretório atual <br>
```man``` - Manual <br>
```ping``` -  <br>
```grep``` -  <br>
```zip``` -  Gera um arquivo compactado zip<br>
```tar``` -  Gera um arquivo compactado tar (mantém as permissões do repositório)<br>
```-h``` - Ajuda<br>
```whoami``` - Informa qual usuário atual<br>
```passwd``` - Dar uma senha para o usuário<br>
```--help``` - Ajuda<br>
```clear``` <br>
```inicio do comando + Tab``` - Completa ou mostra opções para completar o comando <br>
```echo "mensagem"``` - Escreve uma mensagem no terminal<br>
```echo "mensagem" > arquivo.txt``` - Escreve uma mensagem em um arquivo<br>
```echo "mensagem" >> arquivo.txt``` - Adiciona conteúdo no arquivo sem sobrescrever<br>
```cat arquivo.txt``` - Lê um arquivo<br>
```head arquivo.txt``` - Retorna só o início do arquivo (5 primeiras linhas)<br>
```tail arquivo.txt``` - Retorna só o final do arquivo (5 últimas linhas)<br>
