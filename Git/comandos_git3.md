### Tag
```git tag --list```<br>
```git tag -n```

#### Tags em commits antigos
 ```git tag [nome da tag] [codigo do commit]```<br>
 ```git tag -a -m "mensagem" [nome da tag] [codigo do commit]```

#### Enviar tag para repositório
```git push origin V0```
